Project 78 - IK Rig


General Information

Summary: The IK rig created in Project 78 is a heavily altered version of the one from kvoeten. It employs a relative version of the IK rig he created, with much more control and better network syncing.
Coding Platform: Unreal
Target Platform: PCVR
Development Time: 4 weeks

Description

The IK rig was created using C++ and Animation Blueprints. While the initial plugin worked in world space, this one is now in full local space allowing for better syncing and less hanging back. It was created to work in sync with the VR Expansion Plugin and can be used without having to network sync the component.
