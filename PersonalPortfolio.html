My Work

Sand Shader

I created a sand shader that allows for different texture inputs, controllable height and controllable track generation (meaning you can give it different kinds of ways to actually make dents like footsteps or wheel tracks).

It was fully made by me in the Unity Game engine on the Built-In shader pipeline. It currently has a verysimple character controller, that creates footsteps based on speed and walking direction.

Platform: PC
Code Language: C#/Shader
Engine: Unity
My Contribution: All Aspects



Nature Scene

I created a simple dynamic nature scene with several shaders to create a small world. It uses volumetric lighting to create lightrays with a water shader, grass shader and dynamically moving clouds.

The scene itself uses assets from the Unity Asset store. All shader coding, scene layout and lighting was done by me.

Platform: PC
Code Language: Shadergraphs
Engine: Unity
My Contribution: Shader Creation and Scene Setup
